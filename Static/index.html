Link Pod API Documentation

User

Register User

Registers a new user with the system.

Method: POST

Route: /user/create-user

Required Body:
  • email
  • password
Success Response
  • status: true
  • data
    • email
    • _id
    • createdAt
    • updatedAt
Fail Response
  • status: false
  • data: error
  • message: "Server Error"
Login User

Login user.

Method: POST

Route: /user/login

Required Body:
  • email
  • password
Success Response
  • status: true
  • data:
    • email
    • _id
    • createdAt
    • updatedAt
    • name
    • picture
  • message:Success
Fail Response
  • status: false
  • data: error
  • message: "Server Error"
Logout User

Logs out the currently logged-in user.

Method: GET

Route: /user/logout

No body required.

Register With LinkedIn

Registers a new user with LinkedIn authentication.

Method: POST

Route: /user/linkedIn/authorize

Required Body:
  • email
  • password
