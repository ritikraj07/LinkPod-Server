Link Pod API Documentation

User

Register User

Registers a new user with the system.

Method: POST

Route: /user/create-user

Required Body:
  • email
  • password
Success Response
  • status: true
  • data
    • email
    • _id
    • createdAt
    • updatedAt
Fail Response
  • status: false
  • data: error
  • message: "Server Error"
Login User

Login user.

Method: POST

Route: /user/login

Required Body:
  • email
  • password
Success Response
  • status: true
  • data:
    • email
    • _id
    • createdAt
    • updatedAt
    • name
    • picture
  • message:Success
Fail Response
  • status: false
  • data: error
  • message: "Server Error"
Logout User

Logs out the currently logged-in user.

Method: GET

Route: /user/logout

No body required.

Route: /pods/edit/:id

Success Response
  • status: true

POST

Create Post

Registers a new post with the system.

Method: POST

Route: /post/create

Required Body:
  • title
  • description
  • post_url
  • pod_id
  • avgTime (optional) in milliseconds || formate "100000:900000"
Success Response
  • status: true
  • data
    • _id
    • title
    • description
    • urn
    • created_by
    • createdAt
    • updatedAt
Fail Response
  • status: false
  • data: error
  • message: "Internal Server Error"
Get Post by ID

Success Response

Fail Response
